coroutine function

: 함수 실행 도중에 실행 주도권을 다른 함수에 이양 가능하고
: “내가 원하는 시점에” 다시 실행 주도권을 가져올 수 있는 함수를 coroutine 함수라 함.
(non-preemptive하게 실행 가능)
->주도권을 주고 받을 수 있다는 뜻.
->yield를 만나는 순간 stackframe를 유지한채로 주도권을 다른 함수에 넘김. 양보한다? yield~
->파이썬 coroutine은 구현 기반이 generator
->그럴려면 stackframe이 여러개 있어야 될텐데, 이것은 heap에 저장되어 있기에 가능한 일~

함수가 끝나면 stopiteration을 반환해야하는데, 이 객체의 value라는 instance member 아래 리턴값을 넣어서 반환함.

try:
next(g)
except StopIteration as exc:
