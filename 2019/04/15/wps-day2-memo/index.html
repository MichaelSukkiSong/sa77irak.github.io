#bubble sort

def bubble_sort(li):

n = len(li)

for i in range(n-1):

    for j in range(n-1-i):

        if li[j] > li[j+1]:

            li[j], li[j+1] = li[j+1], li[j]

n=4일 때,

총 3(=4-1)번 돌아야 함

j의 값

1번째->3번 0 1 2

2번째->2번 0 1

3번째->1번 0

i=0 j=3 = 4-1-0

i=1 j=2 = 4-1-1

i=2 j=1 = 4-1-2
